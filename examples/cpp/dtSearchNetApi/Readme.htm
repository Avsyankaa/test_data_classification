This project provides source code to dtSearchNetApi.dll, the .NET interface wrapper around the dtSearch Text Retrieval Engine.  To ensure that relative path references are correct, this folder should go here:

C:\Program Files\dtSearch Developer\examples\cpp\dtSearchNetApi

For API documentation, see:

C:\Program Files\dtSearch Developer\help\dtSearchNetApi.chm

dtSearchNetApi.dll relies on dten600.dll, so dten600.dll must either be in a folder that is on the system PATH, or it must be in the same folder as dtSearchNetApi.dll.
